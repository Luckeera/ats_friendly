# Projeto: Certix – Sistema de Gestão e Emissão de Certificados

**Stack Técnica:** Python (Django & REST Framework), React, TypeScript, Vite, PostgreSQL, Vitest e Pytest.

### Descrição Profissional (Pronta para o Currículo)

*   **Arquitetura Full-stack & Dashboard:** Desenvolvi, em conjunto com uma equipe multidisciplinar composta por 5 pessoas, uma plataforma administrativa robusta para gestão de certificados acadêmicos, implementando rotas dinâmicas e interfaces reativas com React e TypeScript para otimizar o fluxo de trabalho dos coordenadores.
*   **Implementação de TDD (Test-Driven Development):** Fui o responsável por estabelecer e executar a cultura de TDD no projeto, garantindo a integridade do sistema através da criação de suítes de testes unitários e de integração tanto no Front-end (Vitest/React Testing Library) quanto no Back-end (Pytest), resultando em uma redução drástica de débitos técnicos e bugs em produção.
*   **Desenvolvimento de APIs RESTful:** Projetei e implementei endpoints escaláveis utilizando Django REST Framework (DRF) para o gerenciamento de emissões, aplicando filtros complexos e modelagem relacional para permitir o rastreio detalhado de dados de alunos (matrícula, e-mail e data de emissão).
*   **Integração de Serviços:** Arquitetura da camada de consumo de APIs no Front-end, utilizando padrões de serviço (Service Pattern) para separar a lógica de negócio da interface, facilitando a manutenção e garantindo a consistência dos dados exibidos no dashboard em tempo real.
*   **Gestão de Fluxo de Dados:** Otimizei a visualização de dados complexos ao criar telas de detalhamento de certificados que consolidam informações de múltiplas entidades, melhorando a capacidade de auditoria e controle de certificados emitidos.
